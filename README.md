## Overview

Clean, fast, and responsive website project. Built with a modern TypeScript stack for optimal performance and developer experience.

## Tech Stack

- **Framework:** Next.js
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Deployment:** Vercel

## Getting Started

```bash
git clone https://github.com/shekelstrong/ermak-site.git
cd ermak-site
npm install
npm run dev
```
